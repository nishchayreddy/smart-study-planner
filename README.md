#  Smart Study Planner

##  Overview

This project is a web-based study planner that helps students organize their study schedule based on subject difficulty and deadlines.

---

##  Features

* Input subjects, deadlines, and difficulty
* Generates a day-wise study plan
* Priority-based scheduling logic
* Task completion tracker (checkbox system)

---

##  Tech Stack

* Python (Flask)
* HTML, CSS, JavaScript

---

##  How to Run

1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000
